How It Works

  1. Install: Set up the encryption tool on your system (one-time setup)
  2. Generate Keys: Create your public/private key pair
  3. Encrypt: Encrypt files using recipient's public key
  4. Upload: Upload encrypted files to SharePoint/OneDrive
  5. Share: Send download link to recipient via secure channel
  6. Download: Recipient downloads the encrypted file
  7. Decrypt: Recipient decrypts using their private key
